sethfischer's dotfiles
======================

Install
-------

```sh
git clone https://github.com/sethfischer/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./bootstrap
```

This will symlink the appropriate files in `.dotfiles` to your home directory.
